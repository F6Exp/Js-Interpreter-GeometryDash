# Geometry Dash - Made by EXP

A mini **Geometry Dash clone** for small display devices using JavaScript.  
Dodge spikes, jump on platforms, avoid obstacles, and try to get the highest score!  

![Preview Image](screenshot.png)

---

## Features

- Classic Geometry Dash gameplay
- Randomly spawning floor spikes, platforms, and obstacles
- Smooth jump and gravity physics
- Score tracking
- Game Over screen
- Start screen with **"Made by EXP"** title

---

## How to Play

1. Use the **Select button** to jump.
2. Avoid spikes and obstacles.
3. Reach the highest score you can!
4. Press Select to restart after Game Over.
5. Press Esc to quit the game.
